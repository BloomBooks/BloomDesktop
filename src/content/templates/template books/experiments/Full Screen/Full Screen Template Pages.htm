You can make a page with whatever layout you need by creating a page from this template page, then doing a "Change Layout" to customize it. Only one image is allowed, and it will fill the whole screen. If you need white text, choose the "whiteText" character style.

To position text, either make empty layout sections that can be resized, or get rid of this text box and instead use the Comic Tool to add bits of text that can be dragged around on top of the image.
