The **Sign Language** template is a collection of pages that have a video frame as well as the usual pictures and text.

There are two ways to get sign language pages into your book.
 1. Use this template to create a book and add more pages from this template as needed.
 2. Modify an existing page using the Change Layout slider at the top and create space for a video spot on the page.

If you are modifying a shell book to include sign lanugage, this second method will help you. If the shell book is locked, you will need to unlock it first in order to customize the page layout for your needs.
