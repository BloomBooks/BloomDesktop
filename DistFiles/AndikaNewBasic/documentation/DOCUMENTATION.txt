DOCUMENTATION
Andika New Basic
========================

This file describes the documentation files included with the Andika 
New Basic font family. This information should be distributed along with the 
Andika New Basic fonts and any derivative works.

Primary documentation for the Andika New Basic font family is on the Andika 
New Basic website (http://scripts.sil.org/Andika_New_Basic).

The AndikaNewBasic-features.pdf file illustrates the use of OpenType and 
Graphite features.

The AndikaNewBasic-features.odt file illustrates the use of OpenType and 
Graphite features (source file for the pdf).
