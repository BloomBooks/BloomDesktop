FONTLOG for the Bloom Show Inv font

This file provides detailed information on the Bloom Show Inv Font Software.
This information should be distributed along with the Bloom Show Inv fonts
and any derivative works.

Basic Font Information

Bloom Show Inv is an open font used by Bloom (https://bloomlibrary.org/)
for showing invisible characters in text. Font sources are in UFO3 format.

ChangeLog

11 July 2023 (SIL font development team) Bloom Show Inv version 2.000
- Revised version with more glyphs and all encoded in the PUA

5 July 2023 (SIL font development team) Bloom Show Inv version 1.000
- Initial version with only nbspace and nnbsp encoded
