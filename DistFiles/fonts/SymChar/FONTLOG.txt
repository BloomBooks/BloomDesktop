FONTLOG for the SymChar fonts

This file provides detailed information on the SymChar Font Software.
This information should be distributed along with the SymChar fonts
(derived from the Bloom Show Inv and Keymanweb-OSK fonts) and any derivative works.

Basic Font Information

SymChar is an open font that can be used by applications to show symbols
that represent 'invisible' characters (such as NBSP). Font sources are in UFO3 format.

ChangeLog
