This folder contains a subset of the files distributed with GraphicsMagick
for Windows from the installer GraphicsMagick-1.3.35-Q16-win32-dll.exe.
Only those files needed for GraphicsMagick to run and to perform the
operations needed by Bloom are included.
Linux users will have the complete GraphicsMagick available because it is
listed as a dependency for bloom-desktop and thus installed automatically.
